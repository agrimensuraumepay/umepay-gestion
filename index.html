Configuración
Conexión con Google Sheets
1
Crear la Google Sheet
Entrá a drive.google.com con la cuenta agrimensuraumepay@gmail.com. Creá una hoja nueva y llamala "Umepay - Gestión".
2
Pegar el Apps Script
En la Sheet: Extensiones → Apps Script. Borrá todo el código y pegá el contenido del archivo umepay_apps_script.gs que te entregamos. Guardá con Ctrl+S.
3
Publicar como Web App
Clic en Implementar → Nueva implementación
• Tipo: Aplicación web
• Ejecutar como: Yo (agrimensuraumepay@gmail.com)
• Quién tiene acceso: Cualquier usuario
Clic en Implementar, autorizá los permisos y copiá la URL que aparece.
4
Pegar la URL acá abajo
Pegá la URL del Web App aquí. Se guarda en tu navegador automáticamente.
